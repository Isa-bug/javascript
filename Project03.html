    1. Identify the three major types of nodes found in an HTML document, and a unique property of each.

    Element Nodes: this is a parent node to all other node types, most common root element node being “html”

    Text nodes: These nodes contain the actual text of an HTML document and can be found inside element nodes. They are not enclosed by any tags.

    Comment nodes: These nodes contain comments that are used by authors to add notes to their code.

    2. Why is the JavaScript textContent property safer than innerHTML?

    innerHTML does a full overwrite, thus removing previous content, the JavaSccript textContent property all symbols are treated as is. The JavaScript textContent property is safer security wise as it only allows for text changes versus structure changes allowed by innerHTML.

    3. Explain the relationship between an HTML attribute and a corresponding DOM property (explain the connection to JavaScript). Use a specific example in your answer.

    Attributes describe properties and values of elements. A “class” attribute is used for the class name of an element, while an “href” attribute is used for a link URL to an anchor. In JavaScript a DOM property is created for each attribute to access and change the attribute’s value.
